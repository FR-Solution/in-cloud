Ресурс ProjectQuota

ProjectQuota — введенная нами абстракция определяющая общую квоту проекта и остаток после пересчета.

Поля ресурса​

  • Описание
  • Структура
  • Ограничения
  • Валидация
  • spec - контент, определяющий проект.
  • spec.quotas - квоты на ресурсы проекта, разделенные на лимиты и запросы для CPU, RAM и DISK.
  • spec.quotas.k8s - квоты на ресурсы проекта, разделенные на лимиты и запросы для CPU, RAM и DISK для K8S.
  • spec.quotas.k8s.limits - лимиты для CPU, RAM в проекте.
  • spec.quotas.k8s.limits.cpu - лимиты запросов на CPU.
  • spec.quotas.k8s.limits.memory - лимиты запросов на RAM.
  • spec.quotas.k8s.requests - запросы для CPU, RAM в проекте.
  • spec.quotas.k8s.requests.cpu - запросы ресурсов на CPU.
  • spec.quotas.k8s.requests.memory - запросы ресурсов на RAM.
  • spec.quotas.k8s.requests.storage - запросы ресурсов на DISK.
  • status - структура, определяющая текущее состояние проекта.
  • status.quotasLeft - остаток квоты на ресурсы проекта, на лимиты и запросы после пересчета квоты.
  • status.quotasLeft.k8s - остаток квоты на ресурсы проекта, на лимиты и запросы для K8S после пересчета квоты.
  • status.quotasLeft.k8s.limits - остаток лимитов для CPU, RAM в проекте после пересчета квоты.
  • status.quotasLeft.k8s.limits.cpu - остаток лимитов по CPU после пересчета квоты.
  • status.quotasLeft.k8s.limits.memory - остаток лимитов по RAM после пересчета квоты.
  • status.quotasLeft.k8s.requests - остаток запросов для CPU, RAM в проекте после пересчета квоты.
  • status.quotasLeft.k8s.requests.cpu - остаток запросов на CPU после пересчета квоты.
  • status.quotasLeft.k8s.requests.memory - остаток запросов на RAM после пересчета квоты.
  • status.quotasLeft.k8s.requests.storage - остаток запросов на DISK после пересчета квоты.
