initAllCerts

All INIT Certificates
  • Kubeadm

Если требуется создать сразу все сертификаты, то воспользуйтесь следующими командами

kubeadm init phase certs all \
    --config=${BASE_K8S_PATH}/kubeadm.conf
kubeadm init phase kubeconfig all \
    --config=${BASE_K8S_PATH}/kubeadm.conf
