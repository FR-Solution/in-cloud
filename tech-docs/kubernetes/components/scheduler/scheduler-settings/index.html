5.2.1.3.7. Kube-Scheduler

warning

Обратите внимание: данный раздел является опциональным и предназначен только для случаев, когда необходимо настроить данный ресурс отдельно от остальных.

  • Static Pod
  • Kubeadm

Переменные окружния

export BASE_K8S_PATH="/etc/kubernetes"
export KUBERNETES_VERSION="v1.30.4"
export KUBE_SCHEDULER_PORT=10259

Рабочая директория

mkdir -p ${BASE_K8S_PATH}/manifests

Генерация манифеста

cat <<EOF > ${BASE_K8S_PATH}/manifests/kube-scheduler.yaml
apiVersion: v1
kind: Pod
metadata:
  creationTimestamp: null
  labels:
    component: kube-scheduler
    tier: control-plane
  name: kube-scheduler
  namespace: kube-system
spec:
  containers:
  - command:
    - kube-scheduler
    - --authentication-kubeconfig=${BASE_K8S_PATH}/scheduler.conf
    - --authorization-kubeconfig=${BASE_K8S_PATH}/scheduler.conf
    - --bind-address=0.0.0.0
    - --kubeconfig=${BASE_K8S_PATH}/scheduler.conf
    - --leader-elect=true
    image: registry.k8s.io/kube-scheduler:${KUBERNETES_VERSION}
    imagePullPolicy: IfNotPresent
    livenessProbe:
      failureThreshold: 8
      httpGet:
        path: /healthz
        port: ${KUBE_SCHEDULER_PORT}
        scheme: HTTPS
      initialDelaySeconds: 10
      periodSeconds: 10
      timeoutSeconds: 15
    name: kube-scheduler
    resources:
      requests:
        cpu: 100m
    startupProbe:
      failureThreshold: 24
      httpGet:
        path: /healthz
        port: ${KUBE_SCHEDULER_PORT}
        scheme: HTTPS
      initialDelaySeconds: 10
      periodSeconds: 10
      timeoutSeconds: 15
    volumeMounts:
    - mountPath: ${BASE_K8S_PATH}/scheduler.conf
      name: kubeconfig
      readOnly: true
  hostNetwork: true
  priority: 2000001000
  priorityClassName: system-node-critical
  securityContext:
    seccompProfile:
      type: RuntimeDefault
  volumes:
  - hostPath:
      path: ${BASE_K8S_PATH}/scheduler.conf
      type: FileOrCreate
    name: kubeconfig
status: {}
EOF

Переменные окружния

export BASE_K8S_PATH="/etc/kubernetes"

Генерация манифеста

kubeadm init phase  control-plane scheduler --config=${BASE_K8S_PATH}/kubeadm.conf
примечание
#### Kube API
[control-plane] Creating static Pod manifest for "kube-scheduler"
