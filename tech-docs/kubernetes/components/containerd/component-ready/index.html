5.2.1.4.1. Containerd

5.2.1.4.1.1. Этапы проверки​

  • Исполняемые файлы существуют.
  • Конфигурационные файлы существуют.
  • Версия исполняемого файла корректна.
  • Статус Systemd Unit.
