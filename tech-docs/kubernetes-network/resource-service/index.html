5.2.5.1.1.2. ClusterIP​

ClusterIP — это базовый тип ресурса Service, который предоставляет виртуальный IP-адрес (VIP) для маршрутизации трафика внутри кластера Kubernetes и обеспечивает точку балансировки для внутренних подключений.

Основные характеристики:​

  • Сервисный адрес доступен только внутри кластера.
  • Виртуальный IP-адрес (VIP) назначает kube-controller-manager (service-cidr-controller).
  • Диапазон сервисных IP-адресов задается статически в параметрах запуска kube-apiserver.
примечание

service-cidr-controller отвечает за управление диапазоном IP-адресов (Service CIDR) для сервисов в кластере.

Функции:​

  • Генерирует виртуальные IP-адреса (ClusterIP) для сервисов.
  • Работает с диапазоном, заданным в kube-apiserver (--service-cluster-ip-range).
  • Следит за выделением IP-адресов, предотвращая дублирование.
