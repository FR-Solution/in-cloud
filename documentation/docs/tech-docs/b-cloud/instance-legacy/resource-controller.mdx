# Контроллер
`Legacy controller` — это контроллер, разработанный для интеграции событийной модели Kubernetes (K8S) с внешними продуктами, такими как Vault, Harbor и другие. Главная цель этого контроллера — снизить нагрузку на разработчиков контроллеров, предоставляя возможность описывать конфигурацию на привычном языке Terraform.
Так же контроллер осуществляет сохранение исходной родительсвой сущности `Insatance` в систему хранения версий, в данном случае в Gitlab
