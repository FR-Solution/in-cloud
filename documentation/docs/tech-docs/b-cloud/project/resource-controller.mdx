# Контроллер
`Контроллер проекта` — это контроллер, разработанный для реализации ряда функций, связанных с ресурсом Project. Основные задачи, которые он выполняет, включают создание базовых ресурсов K8S, необходимых пользователю для взаимодействия с системой.

## Архитектура

## Функции
- Создание пространства имен
- Создание ролевой модели
- Предоставление доступа к пространству имен пользователям через `Group`
- Управление квотами
